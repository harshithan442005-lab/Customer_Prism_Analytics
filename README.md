Customer Prism Analytics

Overview

Customer Prism Analytics is an end-to-end Data Analytics project designed to transform raw customer data into meaningful business insights. The project involves data extraction, cleaning, exploratory data analysis (EDA), SQL-based querying, and interactive dashboard development. By leveraging Python, SQL, PostgreSQL/MySQL/SQL Server, and Power BI, the project provides a comprehensive view of customer demographics, purchasing behavior, revenue trends, product performance, and subscription insights.

Dataset

The dataset contains customer-related information, including:

- Customer demographics (Age Group, Gender)
- Subscription details
- Product categories
- Sales and revenue metrics
- Customer ratings and feedback

The data was used to analyze customer behavior and identify key business trends.

Tools & Technologies

- Python – Data loading, cleaning, and exploratory data analysis
- Pandas & NumPy – Data manipulation and preprocessing
- Matplotlib & Seaborn – Data visualization during EDA
- SQL – Data querying and analysis
- PostgreSQL / MySQL / SQL Server – Database management
- Power BI – Interactive dashboard development

Project Workflow

1. Data Loading

- Imported the dataset into Python.
- Performed initial inspection of data structure and quality.

2. Data Cleaning

- Handled missing values.
- Removed duplicates.
- Corrected data inconsistencies.
- Standardized data formats for analysis.

3. Exploratory Data Analysis (EDA)

- Analyzed customer demographics.
- Explored revenue and sales trends.
- Identified top-performing product categories.
- Examined customer ratings and subscription patterns.

4. Database Integration

- Loaded cleaned data into PostgreSQL/MySQL/SQL Server.
- Created database tables and managed data storage.
- Executed SQL queries to extract business insights.

5. Dashboard Development

- Connected Power BI to the database.
- Designed an interactive analytics dashboard.
- Created KPIs and visualizations for business decision-making.

Key Results

- Identified customer segments based on demographic attributes.
- Analyzed revenue distribution across customer groups.
- Evaluated product category performance.
- Assessed subscription trends and customer engagement.
- Developed an interactive dashboard for monitoring business performance.

Dashboard Features

- Customer Demographics Analysis
- Revenue & Sales Performance Tracking
- Product Category Insights
- Subscription Status Analysis
- Customer Rating Analysis
- Interactive Filters and KPIs

How to Run

Prerequisites

- Python 3.x
- PostgreSQL/MySQL/SQL Server
- Power BI Desktop

Steps

1. Clone this repository.
2. Install required Python libraries.
3. Load the dataset into Python.
4. Perform data cleaning and EDA.
5. Import the cleaned data into PostgreSQL/MySQL/SQL Server.
6. Execute SQL queries for analysis.
7. Connect Power BI to the database.
8. Open the Power BI dashboard and refresh the data.

Project Outcome

This project demonstrates the complete data analytics lifecycle, from data preparation and database management to business intelligence reporting. It highlights practical skills in Python, SQL, database systems, and Power BI while delivering actionable customer and revenue insights.
